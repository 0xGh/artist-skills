# Create Curated Gallery

This skill is for creating or updating a simple one-page curated gallery of Manifold listings.

Its core goal is to help curators promote artists while earning a potential 6.9% referral fee when a listing sale happens through gallery links.

Note: only auction listings are supported at the moment.

## Recommended Gallery Network

The gallery allows to configure an optional `recommendedGallery` link, which lets one curator link to another curator's gallery and automatically passes `?ref=` with their wallet.

?ref= is the curator incentive link: when Gallery A sends traffic to Gallery B with ?ref=<wallet>, Gallery B uses that wallet on 50% of listing referral links, so curators are rewarded for linking to other galleries.

When galleries recommend each other this creates a network of galleries that cross-promote artists and share referral upside across curators.
