# Create Curated Gallery

This skill is for creating or updating a simple one-page curated gallery of Manifold listings.

It helps curators publish a lightweight gallery that promotes artists and can optionally embed Manifold listing widgets.

Note: only auction listings are supported at the moment.

## Recommended Gallery

The gallery allows an optional `recommendedGallery` link.

When set this adds a simple "See More Artworks" link at the bottom of the page so curators can point visitors to another gallery.
